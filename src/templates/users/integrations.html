Jellyseerr Integration

Use a Jellyseerr webhook to automatically add requested/approved media into Shelve.

Jellyseerr Settings

{% csrf_token %}

Controls whether Jellyseerr webhooks are allowed to add items to your Shelve list.

Examples: PENDING,PROCESSING,PARTIALLY_AVAILABLE,AVAILABLE. Leave blank for default behaviour (UNKNOWN is ignored).

Leave blank to allow any requester. If set, only those usernames can trigger adds.
